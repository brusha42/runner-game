# Runner-game

## Описание
- Бесконечный раннер с увеличением скорости движения со временем
- Несколько типов препятствий, наносящие разный урон
- Система бонусов (восстановление очков здоровья, временная неуязвимость)

## Выполненный функционал
### Движение игрока
- экспоненциальное ускорение движения вперед со временем ($currentSpeed = baseSpeed * acceleration^{time}$)
- боковое движение с зависимостью от основной скорости
- система прыжков и скольжения для уворачивания от препятствий
- управление осуществляется на кнопки [A] (влево), [D] (вправо), [Space] (прыжок), [Ctrl] (скольжение)
### Система бонусов
- Щит, дающий временную неуязвимость от препятствий на некоторое время (2с)
- Сердце, восстанавливающее очки здоровья (1), всего у игрока 3 очка здоровья
### Препятствия
- Барьеры, при столкновении с которыми игрок моментально умирает:
    - Обычные высокие стены, которые нельзя перепрыгнуть, но можно оббежать.
    - Низкие стены. Можно перепрыгнуть, но располагаются на всю ширину дороги.
    - Ворота. Под ними можно лишь проскользить.
- Шипы. Если игрок наткнулся на шипы, то теряет очки здоровья (1) и может бежать дальше.
### User Interface
В правом верхнем углу экрана содержится игровая информация:
- Лучший счет игрока за все забеги
- Счет в текущем забеге
- Количество очков здоровья
- Наличие неуязвимости
### Реализация
Основной сценой является `SampleScene`. На ней находится сам объект игрок, а также несколько стартовых дорог.
Во время игры игрок начинает двигаться вперед с некоторой начальной скоростью, увеличивающейся со временем.
Когда игрок пробегает дорогу, то эта дорога уничтожается со сцены и на расстоянии 4х дорог от него спавнится случайная дорога с препятствиями и бонусами. Логика прописана в скрипте `RoadSpawner`.
Скрипт `PlayerHealth` реализует логику, связанную с очками здоровья игрока:
- У препятствий барьеров стоит тег `Obstacle`, таким образом реализована логика моментальной смерти и перезапуска игры.
- Для препятсвий шипов есть свой скрипт. При столкновении с игроком вызывается функция `TakeDamage(spikesDamage)` из `PlayerHealth`.
- При подборе бонуса здоровья (скрипт `HeartBonus`) вызывается функция `Heal(heartHeal)` из `PlayerHealth`.
- При подборе бонуса щита (скрипт `ShieldBonus`) вызывает функция `MakeInvulnerable(duration)` из `PlayerHealth`, которая начинает корутину на время duration(с).

Движение игрока использует новую систему ввода. Вся логика ускорения движения, прыжков и скольжения реализована в скрипте `PlayerMovement`.
